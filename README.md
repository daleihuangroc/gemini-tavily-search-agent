# AI Web Search Agent

This repository contains a Python-based AI agent that leverages Google's Gemini models and the Tavily search API to answer questions with real-time web information.

## Features
- **Autonomous Tool Use**: Automatically searches the web when current information is required.
- **Gemini Integration**: Uses the `gemini-2.5-flash-lite` model.
- **Fast Search**: Powered by Tavily for optimized AI search results.
